# Visor Switch

**🇪🇸** Tu Nintendo Switch en la pantalla de tu PC. Conecta tu capturadora USB HDMI, abre el visor y juega — sin OBS, sin configurar nada.

**🇬🇧** Your Nintendo Switch on your PC screen. Plug in your USB HDMI capture card, open the viewer and play — no OBS, no setup.

## Requisitos / Requirements

- Windows 10 / 11
- Capturadora USB HDMI (probado con chip MS2130 a 1080p 60 fps) / USB HDMI capture card (tested with MS2130 chip at 1080p 60 fps)

## Nota / Note

Si Windows SmartScreen muestra un aviso al abrirlo: **Más información → Ejecutar de todas formas**. Aparece porque el ejecutable no está firmado digitalmente.

If Windows SmartScreen shows a warning: **More info → Run anyway**. It appears because the executable is not code-signed.
